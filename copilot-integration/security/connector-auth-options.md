
# خيارات المصادقة للموصل

## API Key
- بسيط وسريع؛ خزّن القيمة كـ Environment Variable (Secret) واستخدمها في Security.

## OAuth 2.0 (Microsoft Entra ID)
- مناسب للمؤسسات؛ يدعم SSO وتجربة مستخدم أفضل.

مراجع:
- Use environment variables in solution custom connectors – Microsoft Learn.
- Connect external sources via SSO – مذكّرة مركز الرسائل.
