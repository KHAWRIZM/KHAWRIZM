
# تصميم Actions داخل Copilot Studio

1) افتح الوكيل (Agent) → **Tools** أو داخل Topic: **Add an action**.
2) اختر **Connector** وحدّد موصل CometX ثم العملية (مثل CreateTicket).\[1]
3) اربط المدخلات بمتحولات Topic وأعد المخرجات للرد.
4) (اختياري) فعّل **Generative orchestration** ليختار الوكيل الإجراء تلقائيًا.\[2]
